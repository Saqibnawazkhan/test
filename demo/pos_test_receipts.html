TaxNet — POS Test Receipts

Open this on a laptop/second screen (or print it). In the app: Citizen → Services → Verify Receipt → Scan, then point the camera at a code below.

What to expect: a valid FBR invoice is exactly 13 digits → the app shows Verified ✓. Codes with letters or too few digits are fake/unreported → the app shows Not verified and offers “Report this shop”. (The green/red tags below are a presenter cheat-sheet — hide this strip if demoing live.)
